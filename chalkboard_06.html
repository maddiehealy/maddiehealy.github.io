Sorting Fruit

For the following sets of JS functions, we work on sorting an assortment of fruits alphabetically, ultimately delivering whatever fruit comes last in the alphabet. Note that each part builds upon the previous one.

Part 1: Initial Sorting

In this first section, we want to determine which inputted fruit comes last in the alphabet. Our fruit array currently lists: 'Watermelon', 'Apple', 'Orange', and 'Kiwi'.

Can our function output the last alphabetical fruit?

Part 2: Testing Success by Adding More

Now that we have experimented alphabetizing the 4 given fruits above, let's test this function further. Let's add "Banana", "Blueberries", "Mango", and "Peach" into the fruit salad we have already made.

If we add in more fruits, is the sorting still successful?

Part 3: Testing with fruits later in the alphabet

This next function sorts: 'Watermelon', 'Apple', 'Orange', 'Kiwi', 'Blueberries', 'Mango', 'Peach', and 'Yuzu' alphabetically.

What about if we add a fruit that starts with a letter that comes after the W in "Watermelon"? Can our sorting function handle that?
